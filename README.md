## وصف المشروع
تطبيق يستخدم الذكاء الاصطناعي لوصف محتوى الصور تلقائيًا باستخدام نموذج BLIP.

## المميزات
- واجهة مستخدم رسومية سهلة
- وصف الصور باستخدام نموذج BLIP
- عرض عدة أوصاف للصورة الواحدة

## المتطلبات
- Python 3.8+
- PyQt5
- Transformers
- Torch
